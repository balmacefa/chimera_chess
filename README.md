# Chimera Chess Academy

## The Pivot
This project has pivoted from a classic and variant-based chess playing platform built on Python/React to an educational platform focused on mid-level chess players, utilizing a modern **Next.js** full-stack architecture.

## Mission
Our goal is not to be just another place to play standard chess. Instead, we are building a training ground for intermediate players to elevate their game through specialized learning tools.

## Key Features
*   **Minigames & Concepts:** Focused, interactive drills isolating specific tactical or positional elements (e.g., knight outposts, pawn structures).
*   **Mastering Positions:** Structured learning paths for different phases of the game:
    *   **Enter Games (Openings):** Building a solid repertoire and understanding the *why* behind early moves.
    *   **Mid-Game:** Navigating complex, turbulent middle games.
    *   **End-Game:** Precise techniques for converting advantages.
    *   **Titled Play:** Deep dives into positions and moves from Grandmasters and other titled players.
*   **Game Theory:** Applying rigorous strategic and mathematical concepts (minimax, evaluation functions) to human calculation and positional understanding.

## Technical Architecture
We are now utilizing a **Next.js (App Router)** architecture, providing both the frontend React components and the backend API routes within a single, cohesive repository.

*   **Framework:** Next.js (React)
*   **Styling:** Tailwind CSS
*   **Language:** TypeScript

## Getting Started
First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the Academy.
